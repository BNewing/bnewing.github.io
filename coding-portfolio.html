To Do List App

Create items and then drag and drop items between the lists, 'To Do' and 'Done'. Clicking on the status label will flip it to the opposite status and move item into that list. The label also reacts to which list its in when dragged and dropped. Maximum of ten items - because if you have more than ten things to do, you should stop adding stuff and start doing stuff ;)



  • Pure JavaScript - no jQuery (wanted to make it harder)
  • Drag and drop functionality (touch and mouse)
  • Pre-processor - Stylus
  • CSS modules
  • Bundled up with Webpack
